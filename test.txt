| Component       | Description                                                                            |
| --------------- | -------------------------------------------------------------------------------------- |
| `f`             | Prefix to denote "feature"                                                             |
| `<category>`    | One of: `trx`, `bal`, `cat`, `cp`, `exp` (to represent feature families)               |
| `<time_window>` | Time frame in `Xm`, `X_Ym`, or `Xm_to_Ym` format (e.g., `3m`, `1_6m`, `1_6m_to_7_12m`) |
| `<entity>`      | What is being measured (e.g., `credit`, `debit`, `balance`, `Director_Transfers`)      |
| `<metric>`      | Statistic computed (e.g., `sum`, `mean`, `count`, `std`, `min`, `max`, `ratio`)        |
| `<modifier>`    | Optional: transformation like `pct_change`, `trend`, or special conditions             |
